Webpage Script Include
""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

Import as ES6 module.
    ..  code-block:: html

        <!DOCTYPE html>
        <html>
            <head>
                <script type="module">
                    import * as timingsrc from "https://webtiming.github.io/timingsrc/lib/timingsrc-v3.js";
                    console.log(`hello world timingsrc version ${timingsrc.version}!`);
                </script>
            </head>
            <body>
            </body>
        </html>
